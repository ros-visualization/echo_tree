pushEchoTreeToServer(self, rootWord)

source code 

Attempts to connect to the Web server defined by the imported HOST and ECHO_TREE_NEW_ROOT_PORT. If successful, pushes the given new root word to that server. The server will in turn create a new JSON tree, and make it available to any interested clients. It is not an error if the connection attempt to the Web server fails,

Parameters:
  • rootWord (string) - new root word from which an EchoTree will be constructed.
Returns:
None if the update to the EchoTree server succeeded, else an Exception object;
