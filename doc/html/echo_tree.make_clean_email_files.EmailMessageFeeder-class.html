Class EmailMessageFeeder

source code

        EmailMessageFeeder

Iterator that feeds one email message after the other to a caller. Input is the root directory of a file system tree containing files that each hold one email message. The optionally passed-in function fileContentCleaner is called with each email.

Create email message iterator. Every call to 'next()' will deliver a string that is one email message. Assumption is that files in mailDirRoot are email messages, one per file.

Parameters:

  • fileContentCleaner (function) - optional function to call with each email message as a string.
